# PROYECTO-ML
Clasificacion de mensajes acusatorios 

Marco Claderon, Maria Cristina Proaño, Victor Vargas

Este proyecto aborda la clasificación automática de mensajes acusatorios utilizando técnicas de Machine Learning. El objetivo es desarrollar un modelo capaz de identificar este tipo de contenido a partir de texto.
Se implementan procesos de preprocesamiento, extracción de características y entrenamiento de modelos supervisados para construir un sistema de clasificación eficiente.
Además, se realiza la optimización de hiperparámetros y la comparación estadística entre diferentes modelos, con el fin de seleccionar la mejor alternativa.
Finalmente, se incluyen visualizaciones y análisis que permiten interpretar el comportamiento del modelo y la separabilidad de los datos.
